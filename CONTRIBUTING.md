# Contributing to the Age Assurance Dashboard

Thanks for helping improve the database! You don't need to write code — we welcome data contributions, corrections, and UI suggestions.

## How to Contribute

### Add a New Regulation
Use the **[New Regulation](https://github.com/patriciayyzhai/age-assurance-dashboard/issues/new?template=new-regulation.yml)** template. Fill in:
- Regulation name, jurisdiction, and current status
- A source URL confirming the regulation
- A brief 2-3 sentence summary

### Fix Incorrect Data
Use the **[Data Correction](https://github.com/patriciayyzhai/age-assurance-dashboard/issues/new?template=data-correction.yml)** template. Tell us:
- Which regulation needs fixing
- What's currently wrong
- What it should say instead, with a source

### Suggest a UI Improvement
Use the **[UI Enhancement](https://github.com/patriciayyzhai/age-assurance-dashboard/issues/new?template=ui-enhancement.yml)** template. Describe:
- Which page or component
- What you'd like changed
- Why it matters

## What Happens Next

1. Your submission is reviewed by the maintainers
2. High-confidence data may be auto-applied; medium-confidence goes to a review queue
3. You'll be tagged in the issue for any follow-up questions

## Tips

- Always include a **source URL** — it helps us verify quickly
- One issue per regulation or change (don't bundle multiple items)
- If you're unsure about jurisdiction or status, just leave a note — we'll figure it out

Thank you for your contribution! 🎉
